##### Description

This is a getter function gets the drone's **temperature** from the barometer.<br />
The sensor reads the droneâ€™s temperature, not the air around it. Default unit is Celcius.

##### Syntax
```get_temperature()```
```get_temperature(C)```
```get_temperature(F)```
```get_temperature(K)```

##### Parameters

None

##### Returns

The temperature of the given unit as a float. Default unit is Celcius.

##### Example Code
###### Python
```python
#Python code
from codrone_edu.drone import *

drone = Drone()
drone.pair()
temperature = drone.get_temperature()
print(temperature)
drone.close()
```
