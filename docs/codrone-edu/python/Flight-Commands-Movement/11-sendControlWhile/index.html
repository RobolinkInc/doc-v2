sendControlWhile()

Description#

Send roll, pitch, yaw, and throttle values to the drone continously for the given duration in milliseconds

Syntax#

sendControlWhile(roll, pitch, yaw, throttle, duration)

Parameters#

roll: An int from -100 to 100 that sets the roll variable. negative is left, positive is right pitch: An int from -100 to 100 that sets the pitch variable. negative is backwards, positive is forwards yaw: An int from -100 to 100 that sets the yaw variable. negative is left, positive is right throttle: An int from -100 to 100 that sets the throttle variable. negative is down, positive is up duration: A duration in milliseconds
