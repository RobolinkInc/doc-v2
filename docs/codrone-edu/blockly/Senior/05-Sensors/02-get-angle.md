##### Description

This function returns the current gyroscope angle measurement for either the x (roll),y (pitch),or z (yaw) axis. 

![CoDrone EDU angles](xyz.jpg)

##### Parameters
![get angle image](get_angle_params.PNG) <br />
**axis**: x, y, z <br />

##### Returns

**angle**: integer in degrees
