# signal_left_on()

##### Description
Flashes both left front and left back LEDs

##### Syntax
```signal_left_on()```<br />

##### Parameters
None

##### Returns
None

##### Example Code
###### Python
```python
#Python code
from zumi.zumi import Zumi 
zumi = Zumi()
zumi.signal_left_on()
```
