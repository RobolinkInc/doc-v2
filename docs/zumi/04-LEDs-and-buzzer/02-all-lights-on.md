# all_lights_on()

##### Description
Turns all LEDs on
the 2 front white leds and the rear 2 leds.


##### Syntax
```all_lights_on()```<br />

##### Parameters
None

##### Returns
None

##### Example Code
###### Python
```python
#Python code
from zumi.zumi import Zumi 
zumi = Zumi()
zumi.all_lights_on()
```
