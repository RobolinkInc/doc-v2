##### Description
Turns off flashing front and back LEDs

##### Syntax
```hazard_lights_off()```<br />

##### Parameters
None

##### Returns
None

##### Example Code
###### Python
```python
#Python code
from zumi.zumi import Zumi 
import time

zumi = Zumi()
zumi.hazard_lights_on()
time.sleep(2)
zumi.hazard_lights_off()
```
