## eye_off() / eyeOff()

##### Description

This function turns off the eye LEDs

##### Syntax
Python:<br />
```eye_off()```<br />

Arduino:<br />
```eyeOff()```<br />

##### Parameters

None

##### Returns

None

##### Example Code
###### Python
```python
#Python code
import CoDrone

def main():
	drone = CoDrone.CoDrone()
	drone.pair()

	# turn off the eyes
	drone.eye_off()
	
if __name__ == '__main__':
	main()

```
###### Arduino
```c
//Arduino code
#include<CoDrone.h>		//header

void setup(){
//open serial and connect
	CoDrone.begin(115200);
	CoDrone.pair(Nearest);

	// turn off the eyes
	CoDrone.eyeOff();
}

void loop(){
	
}
```
