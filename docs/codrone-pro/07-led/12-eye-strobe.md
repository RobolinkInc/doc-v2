##### Description

This function makes the arms to strobe in various colors

##### Syntax
Python:<br />
```eye_strobe()```<br />

Arduino:<br />
```eyeStrobe()```<br />

##### Parameters

None

##### Returns

None

##### Example Code
###### Python
```python
#Python code
import CoDrone

def main():
	drone = CoDrone.CoDrone()
	drone.pair()

	# strobe the eyes
	drone.eye_strobe()
	
if __name__ == '__main__':
	main()

```
###### Arduino
```c
//Arduino code
```
